# SigaDoc-Visual_Editor
Interface Gráfica de arrasta e solta para criação de modelos FreeMarker para o SigaDoc.

### Novidades da Versão:

- Melhorias no funcionamento das condições
- Agora é possível arrastar e soltar em qualquer lugar do container
- Agora é possível mudar campos de posição, com essa mudança você pode adicioanr um campo que estava fora de um grupo ou condição dentro do mesmo
